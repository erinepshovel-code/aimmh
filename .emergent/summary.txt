<analysis>**original_problem_statement:**
The user wants to build a user interface for prompting multiple AI models simultaneously.

**PRODUCT REQUIREMENTS:**
-   **Models:** Support GPT, Gemini, Claude (using a universal key), Grok, DeepSeek, and Perplexity (with user-provided keys).
-   **UI:**
    -   Minimalist dark theme, optimized for mobile phone screens.
    -   Display multiple model responses at once in a vertically stacked, carousel-like view.
    -   Collapsible sidebar for prompt history.
    -   Enter key should create a new line; Ctrl+Enter should send the prompt. A dedicated Enter button for mobile.
-   **Authentication:** Secure user/password (JWT) and Google OAuth login.
-   **Core Features:**
    -   Settings page to manage API keys.
    -   Session state must persist across window/tab changes until manual logout.
    -   Export conversation history to JSON, TXT, and PDF.
    -   Copy, thumbs up/down, audio output, and share buttons for each individual AI response.
-   **Advanced Research Features (for EDCM):**
    -   **Synthesis:** Send one or more model responses to other models as a new prompt.
    -   **Batch Prompting:** Sequentially send a list of prompts to the selected models.
    -   **Context Control:** A Global Context box to prepend text to all prompts.
    -   **Role Assignment:** Assign specific roles to each model.
    -   **Auto-Cascade:** Automatically chain responses between models for a set number of rounds.
    -   **Auto-Export:** A toggle to automatically export the conversation after each interaction.
-   **Agent Zero Integration:**
    -   The app must communicate with the user's Agent Zero (A0) application running locally on a configurable port (e.g., 8787) and eventually a Google Cloud instance.
    -   Expose API endpoints to allow A0 to act as a control plane.
    -   The app should be able to proxy requests to A0 and auto-ingest conversation transcripts to it.
    -   Display an EDCM dashboard with metrics received from A0: Constraint Mismatch Density, Fixation Coefficient, Escalation Gradient, Context Drift Index, Load Saturation Index.
